清空数据库中的所有表数据 #

清空单表数据很简单。

sql
TRUNCATE TABLE 表名;

TRUNCATE TABLE 表名;
1

但是，如果现在有 100 + 张表？你当然不会一个一个的去 TRUNCATE，但 MySQL 又没有提供该功能。那你可以用用下面的方法。

  1. 查询该数据库下的所有表，利用 CONCAT() 函数将 TRUNCATE 语句拼接起来

    shell
    SELECT
  CONCAT('TRUNCATE TABLE ', TABLE_NAME, ';')
FROM 
  information_schema.TABLES
WHERE TABLES_SCHEMA = '数据库名';

    SELECT
  CONCAT('TRUNCATE TABLE ', TABLE_NAME, ';')
FROM 
  information_schema.TABLES
WHERE TABLES_SCHEMA = '数据库名';

  2. 将执行结果复制，直接执行即可
